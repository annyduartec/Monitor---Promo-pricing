# Promo Pricing Monitor

Static dashboard ready to deploy on GitHub Pages or Vercel.

## Structure

- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`

## Local preview

Open `index.html` in your browser, or serve the folder with any static server.

## Deploy options

### GitHub Pages
Push this folder to a GitHub repository and publish from the `main` branch root.

### Vercel
Import the GitHub repository into Vercel and deploy as a static site.
